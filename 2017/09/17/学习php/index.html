自定义网站根目录

为了分离服务和代码，改变网站根目录地址

  1. 改localhost
  • 打开apache目录中的config文件，搜索并修改documentroot和directory，
  • 重启——打开localhost/文件，成功运行。
  1. 改默认网站根目录地址wamp/www
  • 在wamp根目录下找到wampmanager.ini和wampmanager.tpl
  • wampmanager.ini下把www 目录对应的filename改为自定义目录
  • wampmanager.tpl下把${w_wwwDirectory}对应的filename改为自定义目录
  • 重启

多站点配置

wamp服务下管理多个网站配置

  1. 添加站点
    在wamp/bin/apache/Apache…/conf/extra/httpd-vhost.conf
    documentroot：代码路径
    servername：主机名（域名）
  2. 加载http-vhost.conf
    http-vhost.conf 是扩展文件 默认情况下不会加载
    告诉 Apache 打开httpd.conf 搜索 httpd-vhost
    找到 #include conf/extra/httpd-vhosts.conf 包含扩展文件 去掉注释#
  3. 允许外部主机地址访问
    apache 默认拒绝其他外部主机地址访问服务器下的资源 修改成允许访问
    在httpd.conf 搜索 文件夹名 找到
    deny from all（拒绝其他的外部访问） //deny改成allow
    allow from 127.0.0.1 允许127.0.0.1 本机访问 //注释掉
  4. 重启所有服务
  5. 建立网站代码
    在根目录下 创建多个站点的文件夹，分别新建PHP
  6. 在host文件中添加站点
    打开c:/windows/system32/drivers/etc/hosts 添加站点127.0.0.1 域名1&2..
    指向本地localhost 如果碰到test01.com域名时 优先向127.0.0.1请求资源

自定端口号，为了防止被占用（默认是80端口）

打开 httpd.conf
找到 listen 80和servername localhost:80 改成 8080
之前打开localhost就行，改了之后要打开localhost：端口号

创建项目

创建新项目（路径为apache文件夹）和创建已存在的项目（路径为项目文件夹）

快捷键

ctrl+alt+↓ 复制行
ctrl+d 删除行
alt+↑/↓ 互换行
ctrl+/ 和ctrl+shift+/注释
alt+/ 提示助手
end键 跳转到行末
ctrl+f 查找面板
ctrl(+shift)+k 查找下(上)一个
ctrl+h 全文检索，打开搜索替换窗口,可搜索整个工作空间

ctrl+shift+s 保存所有文件
ctrl+shift+l 查看所有的快捷键
preferences-General-Keys 设置快捷键

工具

  1. 大纲：
    Navigate–Show in–Outline（public-绿色-公开方法 protected-黄色-受保护 private-红色-私有）
  2. 查看错误信息：
    查看错误信息：problems是基于整个程序的，错误提示框位于代码编辑框下面，可以查看文件名、路径、代码行号，双击错误可以跳转到错误位置
  3. 断点调试：
    当前页面首次调试，需要进行调试配置；选中-右键-Debug As-Debug Configurations…-PHP CLI Application-PHP Script页面-默认设置-点击Debug
    当前页面再次调试，可以直接运行；选中-右键-Debug As-1 PHP CLI
    调试快捷键：
    F5:单步调试进入函数内部（单步进入）
    F6:单步调试不进入函数内部（跳过）
    F7:由函数内部返回到调用处（跳出）
    F8：下一断点？之前？
  4. 自动上传代码
    文件上传ftp:
    project-右键-properties-automatic upload
    checked enable automatic upload
    remote connection manage
    upload files : on run (在运行时上传) on save(自动上传) manually （不会被上传）

插件

  1. SVN插件
    svn是个规范代码和管理代码版本的工具
    ​打开Zend studio开发工具，点菜单栏的help->Install New Software；
    弹出窗口的 Work with 栏输入：http://subclipse.tigris.org/update_1.8.x，然后点 Add按钮；
    name随意输入，location默认；
    勾选要安装的插件->next->next->同意协议finish；

    team->comment：备注
    team->ravert：回滚/恢复文件；(默认会滚到最新的文件)
    team->update to Revison:回滚至某个版本
    team->show history->compare current with 3 对比不同的版本

  2. emmet插件
    可快速生成HTML代码

三、语法

输出
echo：不是函数，没有返回值，更快。
print：函数，有返回值，所以相对而言如果只是输出 echo。
print_r：用于打印变量的相关信息，通常在调试中使用。

数据类型
数组：
索引数组：
数组的键是整数，从0开始
赋值方式：$arr[0]=’苹果’;array(‘0’=>’苹果’);array(‘苹果’)数组会默认建立从0开始的整数的键
访问内容：$fruit[‘0’]（也可用变量访问）
关联数组
数组的键是字符串
赋值方式：$arr[‘apple’]=’苹果’;array(‘apple’=>’苹果’);
访问内容：$fruit0 = $fruit[‘banana’];
